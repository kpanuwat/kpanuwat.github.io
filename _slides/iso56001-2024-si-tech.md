## HLS ทำให้ ISO 56001 integrate กับ QMS ที่มีอยู่แล้วได้ทันที

<p class="subhead">Common framework เดียวกับ ISO 9001 / 27001 — ไม่ต้องสร้างระบบใหม่ทั้งหมด</p>

<div class="infographic" style="align-items:stretch; flex-direction:column;">

| Clause | ISO 9001 | ISO 27001 | ISO 56001 (เพิ่มเติม) |
|---|:---:|:---:|---|
| 4: Context | ✓ | ✓ | ครอบ innovation ecosystem + partners |
| 5: Leadership | ✓ | ✓ | เพิ่ม innovation ambition + ownership |
| 6: Planning | ✓ | ✓ | innovation portfolio management |
| 7: Support | ✓ | ✓ | knowledge management + culture |
| 9: Evaluation | ✓ | ✓ | innovation KPIs + value realization |
| 10: Improvement | ✓ | ✓ | ปรับผ่าน innovation retrospective |

</div>

<div class="takeaway"><b>★ Takeaway:</b> SI ที่ certified ISO 9001 แล้วสามารถ integrate ISO 56001 โดยขยาย governance เดิม — ไม่ต้องทำ audit system ใหม่ตั้งแต่ต้น</div>

<!-- Speaker: This is a major selling point for SI management — it's an extension, not a replacement. Shared structure means shared audit cycles. -->
